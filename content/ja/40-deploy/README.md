# デプロイ

このセクションでは、ローカルに構築したPipeCDを用いて、アプリケーションをデプロイしていきます。

デプロイまでの流れは以下の通りです。

1. 設定ファイルを編集する
2. 編集した設定ファイルをリモートにpushする
3. Pipedがその変更を検知する
4. Pipedがデプロイを開始する
